# حل تمرینات بخش 3.5

**! تمرین 3.5.1**: در فضای اعداد صحیح نامنفی، کدام یک از توابع زیر سنجه فاصله (distance measure) هستند؟ اگر هستند، آن را ثابت کنید؛ اگر نیستند، ثابت کنید که قادر به ارضای یک یا چند اصل (axiom) نیستند.
(الف) $max(x, y)$ = عدد بزرگتر بین $x$ و $y$.
(ب) $diff(x, y) = |x − y|$ (مقدار مطلق اختلاف بین $x$ و $y$).
(ج) $sum(x, y) = x + y$.

###  تمرین 3.5.1

برای تعیین اینکه آیا یک تابع $d(x, y)$ یک سنجه فاصله بر روی یک فضا است یا خیر، باید چهار اصل زیر را بررسی کنیم:
1. نامنفی بودن: $d(x, y) \ge 0$.
2. هویت نامتمایزها: $d(x, y) = 0$ اگر و فقط اگر $x = y$.
3. تقارن: $d(x, y) = d(y, x)$.
4. نامساوی مثلثی: $d(x, y) \le d(x, z) + d(z, y)$.

فضای مورد بررسی در این تمرین، مجموعه اعداد صحیح نامنفی است.

**(الف) $d(x, y) = max(x, y)$**

بررسی اصول برای $d(x, y) = max(x, y)$ در فضای اعداد صحیح نامنفی:
1. **نامنفی بودن**: $max(x, y)$ همیشه برای اعداد صحیح نامنفی، نامنفی است. پس اصل 1 برقرار است.
2. **هویت نامتمایزها**: آیا $max(x, y) = 0$ اگر و فقط اگر $x = y$؟
   - اگر $x = y$ و هر دو اعداد صحیح نامنفی باشند، تنها حالتی که $max(x, y) = 0$ می‌شود این است که $x = y = 0$. برای هر $x = y > 0$، $max(x, y) = x > 0$. پس این شرط فقط برای $x=y=0$ برقرار است و برای $x=y>0$ نقض می‌شود.
   - اگر $max(x, y) = 0$ باشد، این به معنی $x \le 0$ و $y \le 0$ است. چون فضا اعداد صحیح نامنفی است، تنها امکان $x=0$ و $y=0$ است، که در این حالت $x=y$ برقرار است.
   - اما مشکل در جهت دیگر است: اگر $x=y>0$، آنگاه $max(x,y) = x > 0 \ne 0$. بنابراین، $d(x, y) = 0$ اگر و فقط اگر $x = y$ برقرار نیست.
   **نتیجه**: اصل 2 نقض می‌شود. برای مثال، $d(1, 1) = max(1, 1) = 1 \ne 0$.
   - *نکته اضافه*: حتی اگر شرط "اگر و فقط اگر $x=y$" را نادیده بگیریم و فقط "اگر $x=y$ آنگاه $d(x,y)=0$" را در نظر بگیریم، این تابع باز هم سنجه فاصله نیست زیرا $d(1,1)=1 \ne 0$.

چون اصل 2 نقض می‌شود، $d(x, y) = max(x, y)$ یک سنجه فاصله نیست. نیازی به بررسی اصول 3 و 4 نیست، اما برای کامل بودن می‌توانیم آن‌ها را نیز بررسی کنیم:
3. **تقارن**: $max(x, y) = max(y, x)$. بله، این برقرار است.
4. **نامساوی مثلثی**: آیا $max(x, y) \le max(x, z) + max(z, y)$؟
   - برای مثال، $x=5, y=2, z=3$.
   - $d(5, 2) = max(5, 2) = 5$.
   - $d(5, 3) = max(5, 3) = 5$.
   - $d(3, 2) = max(3, 2) = 3$.
   - $d(x, y) \le d(x, z) + d(z, y) \implies 5 \le 5 + 3 \implies 5 \le 8$. این مثال برقرار است.
   - اما این اصل در حالت کلی نقض می‌شود. برای مثال $x=5, y=5, z=0$:
   - $d(5, 5) = max(5, 5) = 5$.
   - $d(5, 0) = max(5, 0) = 5$.
   - $d(0, 5) = max(0, 5) = 5$.
   - $d(x, y) \le d(x, z) + d(z, y) \implies 5 \le 5 + 5 \implies 5 \le 10$. این مثال هم برقرار است.
   - بیایید یک مثال دیگر در نظر بگیریم: $x=1, y=10, z=5$.
   - $d(1, 10) = max(1, 10) = 10$.
   - $d(1, 5) = max(1, 5) = 5$.
   - $d(5, 10) = max(5, 10) = 10$.
   - $d(x, y) \le d(x, z) + d(z, y) \implies 10 \le 5 + 10 \implies 10 \le 15$. این مثال هم برقرار است.
   - به نظر می‌رسد نامساوی مثلثی برای $max(x,y)$ برقرار است. برای اثبات: $max(x, y) \le x + y$. نامساوی مثلثی می‌گوید $max(x,y) \le max(x,z) + max(z,y)$.
   - فرض کنید $x \ge y$. آنگاه $max(x, y) = x$. نامساوی می‌شود $x \le max(x, z) + max(z, y)$.
   - اگر $z \le x$ و $z \le y$ (غیرممکن چون $x \ge y$): $x \le x + y$.
   - اگر $z \ge x$ و $z \ge y$: $x \le z + z = 2z$.
   - اگر $x \ge z \ge y$: $x \le x + z$.
   - اگر $z \ge x \ge y$: $x \le z + z$.
   - ... اثبات نامساوی مثلثی برای $max(x,y)$ در فضای اعداد صحیح نامنفی کمی پیچیده‌تر است و ممکن است به سادگی برقرار نباشد. با این حال، از آنجایی که اصل دوم نقض شده است، تابع یک سنجه فاصله نیست.

**(ب) $d(x, y) = |x − y|$**

بررسی اصول برای $d(x, y) = |x − y|$ در فضای اعداد صحیح نامنفی:
1. **نامنفی بودن**: $|x − y|$ مقدار مطلق است، بنابراین همیشه نامنفی است. اصل 1 برقرار است.
2. **هویت نامتمایزها**: آیا $|x − y| = 0$ اگر و فقط اگر $x = y$؟
   - اگر $x = y$، آنگاه $|x − y| = |x − x| = |0| = 0$.
   - اگر $|x − y| = 0$، آنگاه $x − y = 0$، که به معنی $x = y$ است.
   - این اصل برقرار است.
3. **تقارن**: $|x − y| = |−(y − x)| = |y − x|$. این اصل برقرار است.
4. **نامساوی مثلثی**: آیا $|x − y| \le |x − z| + |z − y|$؟ این یک خاصیت شناخته شده از مقدار مطلق است و به نام نامساوی مثلثی برای اعداد حقیقی (و در نتیجه اعداد صحیح نامنفی) شناخته می‌شود. اصل 4 برقرار است.

چون تمام چهار اصل برقرار هستند، $d(x, y) = |x − y|$ یک سنجه فاصله بر روی فضای اعداد صحیح نامنفی است. این همان L1-norm در فضای 1 بعدی است.

**(ج) $d(x, y) = x + y$**

بررسی اصول برای $d(x, y) = x + y$ در فضای اعداد صحیح نامنفی:
1. **نامنفی بودن**: اگر $x$ و $y$ اعداد صحیح نامنفی باشند، $x+y$ نیز نامنفی است. اصل 1 برقرار است.
2. **هویت نامتمایزها**: آیا $x + y = 0$ اگر و فقط اگر $x = y$?
   - اگر $x = y$، آنگاه $d(x, x) = x + x = 2x$. تنها حالتی که $2x=0$ می‌شود این است که $x=0$. بنابراین، $d(x, x) = 0$ فقط زمانی که $x=0$.
   - اصل 2 می‌گوید $d(x, y) = 0$ اگر و فقط اگر $x = y$.
   - در اینجا، $d(0, 0) = 0+0 = 0$ و $0=0$. این بخش از اصل برقرار است.
   - اما برای هر $x > 0$، $d(x, x) = x + x = 2x$. چون $x > 0$ و $x$ صحیح نامنفی است، $2x > 0$. بنابراین برای $x=y>0$، $d(x, y) \ne 0$.
   - **نتیجه**: اصل 2 نقض می‌شود زیرا $d(x, x) \ne 0$ برای $x > 0$.

چون اصل 2 نقض می‌شود، $d(x, y) = x + y$ یک سنجه فاصله نیست. نیازی به بررسی اصول 3 و 4 نیست، اما برای کامل بودن می‌توانیم آن‌ها را نیز بررسی کنیم:
3. **تقارن**: $x + y = y + x$. بله، این برقرار است.
4. **نامساوی مثلثی**: آیا $x + y \le (x + z) + (z + y)$؟
   - $x + y \le x + z + z + y$
   - $0 \le 2z$.
   - از آنجایی که $z$ یک عدد صحیح نامنفی است، $2z \ge 0$. بنابراین نامساوی مثلثی برقرار است.
   - *نکته اضافه*: با وجود اینکه نامساوی مثلثی و تقارن برقرار هستند، نقض اصل هویت نامتمایزها کافی است تا تابع یک سنجه فاصله نباشد.

### راه‌حل تمرین 3.5.2
**تمرین 3.5.2**: فاصله L1 و L2 را بین نقاط (5, 6, 7) و (8, 2, 4) پیدا کنید.
فاصله L1 و L2 را بین نقاط $p_1 = (5, 6, 7)$ و $p_2 = (8, 2, 4)$ در فضای سه‌بعدی اقلیدسی محاسبه می‌کنیم.
این نقاط بردارهایی در فضای 3 بعدی اقلیدسی هستند که مولفه‌های آن‌ها اعداد حقیقی (در اینجا صحیح) هستند.

**فاصله L1 (Manhattan distance)**:
این فاصله برابر با مجموع قدر مطلق تفاوت مولفه‌ها در هر بعد است.
$d_{L1}(p_1, p_2) = |5 - 8| + |6 - 2| + |7 - 4|$
$d_{L1}(p_1, p_2) = |-3| + |4| + |3|$
$d_{L1}(p_1, p_2) = 3 + 4 + 3$
$d_{L1}(p_1, p_2) = 10$

**فاصله L2 (Euclidean distance)**:
این فاصله برابر با ریشه دوم مجموع مربع تفاوت مولفه‌ها در هر بعد است.
$d_{L2}(p_1, p_2) = \sqrt{(5 - 8)^2 + (6 - 2)^2 + (7 - 4)^2}$
$d_{L2}(p_1, p_2) = \sqrt{(-3)^2 + (4)^2 + (3)^2}$
$d_{L2}(p_1, p_2) = \sqrt{9 + 16 + 9}$
$d_{L2}(p_1, p_2) = \sqrt{34}$

**نتیجه**:
- فاصله L1 بین نقاط برابر با 10 است.
- فاصله L2 بین نقاط برابر با $\sqrt{34}$ است.

###  تمرین 3.5.3
**!! تمرین 3.5.3**: ثابت کنید که اگر $i$ و $j$ هر دو اعداد صحیح مثبت باشند و $i < j$، آنگاه نرم Li بین هر دو نقطه بیشتر از نرم Lj بین همان دو نقطه است.

**اثبات**: اگر $i$ و $j$ هر دو اعداد صحیح مثبت باشند و $i < j$، آنگاه نرم Li بین هر دو نقطه بزرگتر یا مساوی نرم Lj بین همان دو نقطه است.

اجازه دهید دو نقطه دلخواه $x = [x_1, \dots, x_n]$ و $y = [y_1, \dots, y_n]$ در یک فضای $n$-بعدی اقلیدسی را در نظر بگیریم. فاصله Lr بین این دو نقطه به صورت زیر تعریف می‌شود:
$d_r(x, y) = \left( \sum_{k=1}^n |x_k - y_k|^r \right)^{1/r}$

ما می‌خواهیم ثابت کنیم که اگر $0 < i < j$، آنگاه $d_i(x, y) \ge d_j(x, y)$.
اجازه دهید $v_k = |x_k - y_k|$ برای $k=1, \dots, n$. واضح است که $v_k \ge 0$.
ما باید ثابت کنیم که برای $0 < i < j$:
$\left( \sum_{k=1}^n v_k^i \right)^{1/i} \ge \left( \sum_{k=1}^n v_k^j \right)^{1/j}$

این یک نتیجه استاندارد از نامساوی میانگین‌های تعمیم یافته (Generalized Mean Inequality) است که بیان می‌کند برای اعداد نامنفی $v_1, \dots, v_n$ و $0 < i < j$:
$\left( \frac{1}{n} \sum_{k=1}^n v_k^i \right)^{1/i} \le \left( \frac{1}{n} \sum_{k=1}^n v_k^j \right)^{1/j}$
این نامساوی به صورت $\left( \sum v_k^i \right)^{1/i} / n^{1/i} \le \left( \sum v_k^j \right)^{1/j} / n^{1/j}$ نوشته می‌شود.
ضرب هر دو طرف در $n$ به توان $1/i$ می‌دهد:
$\left( \sum v_k^i \right)^{1/i} \le n^{1/i} \left( \sum v_k^j \right)^{1/j} / n^{1/j} = n^{(1/i - 1/j)} \left( \sum v_k^j \right)^{1/j}$

این نتیجه می‌گوید $L_i \le n^{(1/i - 1/j)} L_j$ برای $i < j$. این دقیقا برعکس آن چیزی است که در صورت سوال آمده است ("Li norm ... is greater than the Lj norm").

با این حال، نامساوی استاندارد دیگری وجود دارد که به طور مستقیم مقایسه توان‌ها را در مجموع‌ها انجام می‌دهد و به نام نامساوی هولدر (یا در موارد خاص، مرتبط با نامساوی میانگین‌های وزنی توان) شناخته می‌شود. برای $0 < i < j$، تابع $f(t) = t^{j/i}$ برای $t \ge 0$ محدب (convex) است، زیرا $j/i > 1$. با استفاده از نامساوی جنسن برای مجموع و توان $j/i > 1$:
اجازه دهید $p = j/i > 1$. ما می‌خواهیم $\left( \sum v_k^i \right)^{1/i}$ را با $\left( \sum v_k^{ip} \right)^{1/(ip)}$ مقایسه کنیم.
$(\sum v_k^j)^{1/j} = (\sum (v_k^i)^{j/i})^{1/j}$.
اگر $\sum v_k^i = S$, آنگاه $v_k^i = S w_k$ که $\sum w_k = 1$ و $w_k \ge 0$.
پس $\sum v_k^j = \sum (v_k^i)^{j/i} = \sum (S w_k)^{j/i} = S^{j/i} \sum w_k^{j/i}$.
بنابراین، ما در حال مقایسه $S^{1/i}$ و $(S^{j/i} \sum w_k^{j/i})^{1/j} = S^{1/i} (\sum w_k^{j/i})^{1/j}$ هستیم.
ما باید ثابت کنیم $1 \ge (\sum w_k^{j/i})^{1/j}$ که معادل $1^j \ge \sum w_k^{j/i}$ یا $1 \ge \sum w_k^{j/i}$ است.
چون $w_k \ge 0$ و $\sum w_k = 1$ و $j/i > 1$، می‌توان از خاصیت تابع $f(w) = w^p$ برای $p>1$ که محدب است استفاده کرد.
برای $p = j/i > 1$:
$\sum w_k^{j/i} \le (\sum w_k)^{j/i} = 1^{j/i} = 1$.
این نشان می‌دهد که $\sum w_k^{j/i} \le 1$.
در نتیجه، $(\sum w_k^{j/i})^{1/j} \le 1^{1/j} = 1$.
این اثبات می‌کند که $S^{1/i} \ge S^{1/i} (\sum w_k^{j/i})^{1/j}$, که معادل $\left( \sum v_k^i \right)^{1/i} \ge \left( \sum v_k^j \right)^{1/j}$ است.

**نتیجه گیری از اثبات**:
برای $0 < i < j$ و اعداد نامنفی $v_k = |x_k - y_k|$، داریم:
$d_i(x, y) = \left( \sum_{k=1}^n v_k^i \right)^{1/i} \ge \left( \sum_{k=1}^n v_k^j \right)^{1/j} = d_j(x, y)$
یعنی نرم Li همیشه **بزرگتر یا مساوی** نرم Lj برای $i < j$ است.

**شرط برابری**:
برابری $d_i(x, y) = d_j(x, y)$ زمانی رخ می‌دهد که:
1. تمام $v_k$ برابر صفر باشند (یعنی $x = y$). در این صورت، هر دو فاصله برابر 0 هستند.
2. تنها یک $v_k$ غیرصفر باشد (یعنی نقاط فقط در یک بعد تفاوت داشته باشند). اگر تنها $v_{k_0} \ne 0$، آنگاه $\left( v_{k_0}^i \right)^{1/i} = v_{k_0}$ و $\left( v_{k_0}^j \right)^{1/j} = v_{k_0}$. هر دو فاصله برابر با $|x_{k_0} - y_{k_0}|$ خواهند بود.

در تمام موارد دیگر که حداقل دو $v_k$ غیرصفر متفاوت وجود دارند (مثلا $v_1=1, v_2=2$), نامساوی اکید است: $d_i(x, y) > d_j(x, y)$ برای $i < j$. صورت سوال می‌گوید "greater than"، که نشان می‌دهد احتمالاً حالت‌های بدیهی (نقاط یکسان یا تفاوت فقط در یک بعد) در نظر گرفته نشده‌اند، یا از اصطلاح "greater than" به جای "greater than or equal to" استفاده شده است. اثبات بالا نشان می‌دهد که در حالت کلی (زمانی که حداقل دو مولفه تفاوت غیر صفر دارند)، نامساوی اکید برقرار است.

### راه‌حل تمرین 3.5.4
**تمرین 3.5.4**: فاصله ژاکارد را بین جفت مجموعه‌های زیر پیدا کنید:
(الف) {1, 2, 3, 4} و {2, 3, 4, 5}.
(ب) {1, 2, 3} و {4, 5, 6}.
فاصله ژاکارد بین دو مجموعه $A$ و $B$ برابر با $1 - SIM(A, B)$ است.
شباهت ژاکارد $SIM(A, B)$ به صورت نسبت اندازه اشتراک دو مجموعه به اندازه اجتماع آن‌ها تعریف می‌شود:
$SIM(A, B) = \frac{|A \cap B|}{|A \cup B|}$

**(الف) مجموعه‌های {1, 2, 3, 4} و {2, 3, 4, 5}**

اجازه دهید $A = \{1, 2, 3, 4\}$ و $B = \{2, 3, 4, 5\}$.
- اشتراک دو مجموعه: $A \cap B = \{2, 3, 4\}$
- اندازه اشتراک: $|A \cap B| = 3$
- اجتماع دو مجموعه: $A \cup B = \{1, 2, 3, 4, 5\}$
- اندازه اجتماع: $|A \cup B| = 5$

شباهت ژاکارد:
$SIM(A, B) = \frac{|A \cap B|}{|A \cup B|} = \frac{3}{5} = 0.6$

فاصله ژاکارد:
$d(A, B) = 1 - SIM(A, B) = 1 - 0.6 = 0.4$

**(ب) مجموعه‌های {1, 2, 3} و {4, 5, 6}**

اجازه دهید $C = \{1, 2, 3\}$ و $D = \{4, 5, 6\}$.
- اشتراک دو مجموعه: $C \cap D = \{\}$ (مجموعه تهی)
- اندازه اشتراک: $|C \cap D| = 0$
- اجتماع دو مجموعه: $C \cup D = \{1, 2, 3, 4, 5, 6\}$
- اندازه اجتماع: $|C \cup D| = 6$

شباهت ژاکارد:
$SIM(C, D) = \frac{|C \cap D|}{|C \cup D|} = \frac{0}{6} = 0$

فاصله ژاکارد:
$d(C, D) = 1 - SIM(C, D) = 1 - 0 = 1$
